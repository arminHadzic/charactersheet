# Model sheet
## What is an animation model sheet?
- To put it simply, a model sheet has multiple drawings of a character that shows specific details, proportions, poses, expressions, and various angles of a character in a particular animation style.
- A model sheet is a comprehensive reference document ensuring consistency across different animators and scenes. It includes a full-body turnaround (front, back, 3/4, side views), detailed expression sheets, mouth charts for lip-syncing, color palettes, and specific poses, props, or costume details.
- See for more detail: https://en.wikipedia.org/wiki/Model_sheet

## Core components of an model sheet:
- The animation style must be consistent across all the drawings in the model sheet, this is crucial.
- Expression Sheet
- Pose sheet
- Character construction sheet
- Turnaround (Rotation) sheet
- Lip-Sync/Mouth Chart: Specific drawings of mouth shapes corresponding to phonemes (sounds) to guide animators during dialogue.
- Color Palette/Reference: The precise colors for the character’s hair, skin, clothing, and eyes to maintain color consistency across scenes.
- Details and Constraints: Specifics about clothing, accessories, or unique markings, and sometimes, "do's and don'ts" to keep the character "on model".

## Model sheet types:
Model sheets can be categorized further into one of the following:
1. Expression sheets: A collection of facial expressions—such as happy, angry, sad, scared—that define how the character conveys emotion.
2. Pose sheets: Illustrations of the character in common or extreme poses that showcase their personality and how they move.
3. Character construction sheets: These sheets focus on the character's physical structure, proportions, construction, and how they are put together.
4. Character turnaround (rotation) sheets: A 360-degree view (usually front, 3/4, profile, back) that shows all character details, with guidelines to ensure proportions remain consistent across angles.

## How to make a model sheet:
1. Start with the character's attributes. These include the character's personality, background, and any other relevant information. As the user for these details.
- For example, a character with a short temper will need an angry expression but also a friendly one to balance the two out. While short temper may be the primary attribute, a couple additional attributes could be confident, lively, dramatic, driven, etc.
2. Draw the various expressions and poses, keeping the attributes in mind. For the initial poses and expressions focus on the character's primary attributes.
- In our example, short tempered person could be expressed as having a hand on their hip and an angled torso, or arms crossed.
- Use clean and clear lines for modle sheets, to help define the character and avoid confusion.
3. In the second drawing of the model sheet, draw the character in a neutral pose, showing the front, back, and side views of the character. This will help the user to see the character from all angles and understand their proportions.
- Make sure to go for a different angle from the first drawing, to convey the character's proportions and shape better.
4. For the third drawing, draw the character in an irregular (but naturally occurring pose) that may be more difficult to draw. This will help the user to see how the character looks in different poses and angles.
- For example, consider poses like the character looking up with their chin tilted up a little, or the character looking down with a look of disgust. High and low angle poses are great for this.
- Note, this drawing does not have to be the full body of the character if one of the previous drawings already shows the character's body from that angle. Try to focus on information quality.
5. Draw additional poses, angles, expression, etc. Take into account physicality of the character, and which poses and expressions would best convey their personality and attributes. Do not crowd the character sheet page, try to keep the drawings properly spaced out. 
6. If there is no room for additional drawings then work on defining the color palette on the side, where we highlight the main colors used in coloring the character. 

## What makes a good model sheet:
- Draw the character from multiple angles and perspectives. E.g., draw the character sitting down sulking, jumping for joy, running, etc.
- Avoid drawing expressions that are not relevant to the character, that is a waste of information.
- Add a size reference if the character is particularly tall, short, big, or small. For example, a banana can be used as a size reference for a small character or a lamppost for a tall character.
- Include color in the modle sheet for at least one of the drawings and pair this with color swatches corresponding to these colors. Do not overload the page, select the most important colors on the character (ignore background colors) and leave plenty of white space around the color swatches.
